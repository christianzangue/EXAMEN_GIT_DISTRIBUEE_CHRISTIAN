Présentation du Dev Mode

Le Dev Mode est une interface dédiée aux développeurs qui permet d'examiner des designs et de les parcourir. Vous pouvez accéder au Dev Mode dans n'importe quel fichier de design Figma. Pour passer du Design Mode au Dev Mode : Ouvrez un fichier de design Figma.
