C'est quoi un serveur git ? Git est un logiciel VCS (Version Control System) local qui permet aux professionnels de la data et aux développeurs de sauvegarder l'historique de modifications de leurs projets. Quant à GitHub, il s'agit d'une plateforme web qui intègre les fonctionnalités de contrôle de versions de Git.
